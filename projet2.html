Projet n°2, Base de donnée Jet-ski en SQL

Voir le code source sur GitHub

Contexte

Lors du deuxième projet de BUT informatique, j’ai dû coder en SQL une Base de donnée qui possédait les fonctionnalités suivantes :
Le script .sql permettant de supprimer, créer, remplir (jeu de données), afficher les tables et les requêtes permettant de répondre aux questions

Objectif

L'objectif était de concevoir et implémenter une base de données complète pour une entreprise de location de jet-skis, incluant la gestion des produits, des clients, des fournisseurs et des locations.
Le projet devait fournir un script SQL permettant de créer la structure de la base, d'insérer un jeu de données de test, et de réaliser des requêtes d'analyse pertinentes pour le métier.

Travail réalisé

J'ai développé une base de données complète qui possédait les fonctionnalités suivantes :
- Création de la base de données et des tables nécessaires à la gestion des jet-skis, clients, fournisseurs et locations.
- Insertion d'un jeu de données de test pour simuler des opérations réelles.
- Requêtes permettant de répondre à des questions métier, telles que la disponibilité des jet-skis, les clients les plus fréquents, etc.
- Gestion des relations entre les tables pour assurer l'intégrité des données.

Résultats

Appréciation : Les premiers retours sont positives malgrés que ce travail n'est pas encore évalué

Compétences développées :
- Conception de bases de données relationnelles
- Rédaction de requêtes SQL "complexes"
- Gestion / analyses des données
- Compréhension des besoins métier et traduction en schéma de base de données

Ce que ce projet m'a appris : Ce projet m'a permis de comprendre l'importance de la modélisation des données et de la gestion des relations entre les tables. J'ai également appris à rédiger des requêtes SQL complexes pour extraire des informations pertinentes à partir de la base de données. Cela m'a donné une meilleure compréhension de la manière dont les bases de données sont utilisées dans le monde réel et comment elles peuvent être optimisées pour répondre aux besoins d'une entreprise.
J'ai particulièrement apprécié le défi de concevoir une base de données complète et fonctionnelle. Cela m'a permis d'appliquer mes connaissances théoriques à un projet concret et de voir comment les bases de données peuvent être utilisées pour résoudre des problèmes réels. J'ai également aimé travailler sur les requêtes SQL, car cela m'a permis d'explorer différentes façons d'interroger et d'analyser les données.
